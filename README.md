#### Description: I have created a website that helps trades (forex,index,metals,futures), to journal and visualise their trades without overcomplicating statistics. The website has 3 main pages, the dashboard("/") which shows a summary of your trades, and basic statistics such as winrate, the month's profit, number of trades this month, average RR, etc. The summary of trades can then be expanded into the trades page ("/trades") which then shows you all of your past trades, which can be filtered and sorted accordingly. The last page is the trade viewing page, which allows users to view the details of their trade and the notes they attached to that specific trade.

#### The project is mostly a server sided program that relies heavily on Postgres to store the database and app.py that handles all the routes.
